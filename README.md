# How to Run This Project?

## Create .env File
```
NEXT_PUBLIC_POKE_API_ENDPOINT="https://pokeapi.co/api/v2"
```

## Using Bun
```
bun install
bun dev
```

## Using Yarn
```
yarn install
yarn dev
```
